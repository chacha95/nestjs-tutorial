# Nestjs-Tutorial

## Installation

```bash
$ yarn
```

## Local Environment

db migration

```bash
# create
$ npm run makemigration "message"

# apply
$ npm run migrate
```

## Running the app

```bash
# development
$ yarn run start:local

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
